Turn-in

Export your schematic to a PDF file in EAGLE and upload the resulting file to Canvas. To do this follow these steps from your schematic.
  • File → Print,
  • In the Print pop-up,
    • Printer: Microsoft Print to PDF
    • Orientation: <whatever best accommodates the layout>
    • Alignment: Center
    • Area: Full
    • Scale factor: <bigget possible to fit a single page>
  • Click OK,
  • In the "Save Print Output As" pop-up, navigate to your project directory and give your file a meaningful name.
