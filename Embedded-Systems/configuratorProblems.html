Problems with the Mcrochips Code Configurator

Problem getting code configurator running in the lab?
  1. Go to https://www.microchip.com/mplab/mplab-code-configurator
  2. Scroll to the bottom of the page, click on "Current Download"
  3. Download MCC v3.66 plugin
  4. Extract the files
  5. Go onto MPLAB X, click on "Tools" and then "Plugins Download"
  6. Click on "Go to MPLAB X Plugin Manager"
  7. Click on "Downloaded" and then "Add Plugins"
  8. Go to the file path of the downloaded plugin and click "open"
  9. Check the plugin and click install
