Lab 3 assignment

Complete the following assignment individually or in teams of at most two.

I need your help to design a creative outreach activity to teach people how to solder and through this increase their interest in Electrical Engineering. For this activity, you will be constrained to used the LED flower circuit provided to you in this inLab. The following are the objectives:
  • Rotate layout to minimize bounding box area,
  • PCB must fit in a 75mm x 100mm bag,
  • Mimic some object,
  • The mimicked object should use color in some logical way,
  • The object should be reasonably gender neutral,
  • Something that your mom would approve of,
  • Solder mask color available from JLC PCB,
  • Place components to enhance appearance,
  • Mines theme?,
  • No sharp points.
Before turning the assignment in, please:
  • Run the DRC with the vuMeter Design Rules rule set provided to you in the previous lab. Clear all warning and errors.
  • Print your layout to PDF as follows:
    • File → Print,
    • In the Print pop-up,
      • Printer: Microsoft Print to PDF
      • Orientation: <whatever best accommodates the layout>
      • Alignment: Center
      • Area: Full
      • Scale factor: <bigget possible to fit a single page>
    • Click OK,
    • In the "Save Print Output As" pop-up, navigate to your lab02 directory and give your file a meaningful name.
  • Upload your PDF to Canvas in the "Lab 3 - RGB layout" assignment.
